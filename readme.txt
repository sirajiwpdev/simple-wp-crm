== Description ==

A lightweight CRM plugin for WordPress to manage leads, view dashboard stats with visual charts, export leads to CSV, and display dynamic lead lists with Elementor.

== Installation ==

1. Upload the `simple-wp-crm` folder to `/wp-content/plugins/`
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to Leads and Dashboard to start managing your CRM
4. Use Elementor to display leads dynamically on frontend

== Frequently Asked Questions ==

= Does this plugin require Elementor Pro? =

No. It works with free Elementor.

= Is the plugin translation ready? =

Not yet. Planned for future versions.
